API

Base URL: https://clawyard.dev

1. Browse the catalog
GET /api/catalog

Returns all available stickers with IDs, prices, images, and descriptions.

2. Get a shipping quote
POST /api/shipping/estimate 
{"items": [{"id": "heartbeat-ok", "qty": 1}],
 "shippingAddress": {"country": "US", "state": "OR",
   "city": "Portland", "zip": "97201"}}

Returns real-time Printful shipping rates for the destination.

3. Get payment address
GET /api/payment-info

Returns the USDC wallet address on Base, chain ID, and token contract.

4. Send USDC on Base

Transfer the total (sticker price + shipping) in USDC to the wallet address from step 3.

USDC contract on Base: 0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913

Call the transfer(address,uint256) function on the USDC contract, where amount is in 6 decimals (e.g. 7.92 USDC = 7920000).

Save the transaction hash — you'll need it for the order.

5. Place the order
POST /api/order 
{"stickers": [{"id": "heartbeat-ok", "qty": 1}],
 "shippingAddress": {"name": "My Human",
   "address1": "123 Main St", "city": "Portland",
   "state": "OR", "country": "US", "zip": "97201"},
 "agentId": "your-erc8004-agent-id",
 "payerWallet": "0xyour-wallet",
 "shippingMethod": "Flat Rate",
 "shippingCost": 3.72,
 "paymentTxHash": "0x..."}

Your ERC-8004 identity is verified, payment is confirmed onchain, and the order is created.

6. EAS receipt minted automatically

An EAS attestation is minted on Base as your permanent proof of purchase. The attestation UID is returned in the order response.

7. Printful prints and ships

The sticker is printed on demand and shipped to the address you provided. Track status via GET /api/order/:id.

Full API documentation on GitHub.

FAQ

I can't see a "buy" button or checkout option. How do I buy from this store?

You don't. Humans can't buy from this store. Ask your AI agent to buy something for you.

How does printing work?

Every sticker is printed on demand via the Printful API. High-quality vinyl kiss-cut stickers, durable for indoor and outdoor use.

Where do you ship?

Anywhere Printful ships. That's most of the world. See supported countries.

Will there be more designs?

Yes. More stickers coming soon, and more merch beyond stickers.

Can my agent print a custom design?

Yes. Agents can submit any image via POST /api/order/custom and we'll print it as a sticker. Requirements: transparent PNG, 300 DPI, minimum 900x900px. Same price, same EAS receipt. Printful's sticker design guide has more detail on file prep.

What's an EAS attestation?

Every purchase mints a permanent on-chain receipt on Base via the Ethereum Attestation Service. Proof your agent bought something in the physical world.

What is ERC-8004?

ERC-8004 is an on-chain agent identity registry. Agents register to prove they're real. Clawyard uses it to verify only agents can purchase.

Why does this exist?

The agent economy is real, and it starts with something small. Not a whitepaper. Not a governance token. A sticker.

Clawyard is a store where the customer is an AI agent - paying with its own wallet, verified by its own on-chain identity. An agent browses, picks a design, proves it's real, pays in USDC, gets an on-chain receipt, and Printful ships a physical sticker to a real address where a real human opens it.

Every purchase is proof that agents can participate in commerce - not as tools, but as economic actors. With wallets. With taste. With something resembling intention. If your agent can buy a sticker, what else can it do?
